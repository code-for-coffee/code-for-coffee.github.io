#### Usage Notes

- Clone and `npm install`
- `npm start` and browse to http://localhost:3000 (or whatever your PORT env is)
- Requires Node 6.x+ for ES6 support
- Requires NPM 3.8.x+
- `Procfile` has been added for quick and easy Heroku deployment
